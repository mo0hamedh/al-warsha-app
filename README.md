## 📱 عن التطبيق
الورشة تطبيق متكامل لتطوير الذات يجمع بين:
- ✅ إدارة المهام اليومية
- 🧠 تتبع جلسات التركيز (Pomodoro)
- 📋 جدول أسبوعي تفاعلي
- 🔥 تتبع العادات الإيجابية والسلبية
- 🏆 منافسة شهرية بين المستخدمين
- 👥 نظام أصدقاء وليدربورد

## 🛠️ التقنيات المستخدمة
- **Flutter** - Cross-platform UI
- **Firebase Auth** - المصادقة
- **Cloud Firestore** - قاعدة البيانات
- **Firebase Storage** - تخزين الصور
- **fl_chart** - الرسوم البيانية
- **Google Fonts** - Cairo & Tajawal

## 🚀 تشغيل المشروع

### المتطلبات
- Flutter SDK 3.x
- Firebase Project
- Dart SDK

### الخطوات
1. Clone المشروع
```bash
git clone https://github.com/mo0hamedh/al-warsha-app.git
```

2. تثبيت الـ packages
```bash
flutter pub get
```

3. إعداد Firebase
- أنشئ مشروع على Firebase Console
- فعّل Authentication و Firestore
- حمّل google-services.json و GoogleService-Info.plist واستبدلها في مسارات المشروعات.

4. تشغيل التطبيق
```bash
flutter run
```
